# dropdownemenu

>

[![NPM](https://img.shields.io/npm/v/dropdownemenu.svg)](https://www.npmjs.com/package/dropdownemenu) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save dropdownemenu
```

## Description

- show dropdown menu for navbar links on click event

## Usage

```jsx
import React, { Component } from "react";

import dropdownemenu from "dropdownemenu";

class Example extends Component {
  render() {
    return <dropdownemenu />;
  }
}
```
